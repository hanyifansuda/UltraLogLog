# UltraLogLog

The C++ version of the cardinality estimation algorithm [UltraLogLog](https://github.com/dynatrace-research/ultraloglog-paper) in VLDB 2024.

## Usage
Please refer to main.cpp and the original paper.

## Reference
Ertl, O., 2024. UltraLogLog: A Practical and More Space-Efficient Alternative to HyperLogLog for Approximate Distinct Counting. Proceedings of the VLDB Endowment, 17(7), pp.1655-1668.
